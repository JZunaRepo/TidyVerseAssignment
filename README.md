## Jonnathan Zuna Largo — TidyVerse Vignette: Data Science Job Postings 2025

**Packages used:** dplyr, tidyr, ggplot2, stringr

**Dataset:** Data Science Job Posts 2025 (Kaggle)

**Description:** This vignette demonstrates how to use core TidyVerse functions 
to clean, transform, and visualize real-world hiring data. Topics covered include 
parsing messy salary strings with stringr, unnesting list-valued skill columns 
with tidyr, aggregating with dplyr's group_by and summarise, and building 
publication-ready charts with ggplot2.
